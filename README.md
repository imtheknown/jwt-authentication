# jwt-authentication

NodeJS Express Project Showcasing login and signup features, validation using Joi, and JWT for authentication of routes.


# Project Structure
```.
├── index.js
├── middlewares
│   └── validation.js
├── model
│   └── user.js
├── package.json
├── README.md
└── routes
    ├── auth.js
    ├── posts.js
    └── verifyToken.js
```

# For running the project
```
yarn install 
yarn start
```
### If you use npm

```
npm install 
npm start
```
